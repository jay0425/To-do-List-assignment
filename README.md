데스크탑, 태블릿, 모바일 반응형으로 구현된 투두리스트 페이지입니다.

# 로고

로고를 클릭하면 리스트페이지로 이동합니다.

# 리스트 페이지

## 할일 추가하기

'할일을 입력해주세요' 칸에 할일을 입력한뒤 엔터를 하거나 "추가하기" 버튼을 클릭하면 할일이 추가됩니다.

## 할일 완료하기

TO DO 뱃지 아래의 할일의 왼쪽 동그라미를 클릭하면
DONE 뱃지 아래로 완료된 할일이 이동됩니다.

## 완료된 할일 다시 복구하기

DONE 뱃지 아래의 완료된 일의 좌측 버튼을 클릭하면 다시 할일로 복구되어 재출력됩니다.

# 상세 페이지

## 제목 버튼

제목의 우측 버튼을 클릭하면 할일의 완료, 진행 상태를 변경할 수 있습니다.

## 디테일 페이지 내용 수정

제목, 메모 내용을 수정할 수 있고 파일 업로드 부분에서 사진을 업로드하고
마지막으로 "수정 완료" 버튼 클릭 시 수정한 내용이 수정 요청되어집니다.
요청 성공 시 리스트 페이지로 이동됩니다.

## 디테일 페이지 삭제

"삭제하기" 버튼 클릭 시 해당 할일이 삭제 되고 리스트 페이지로 이동됩니다.
